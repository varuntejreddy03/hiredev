# HireDev — IT Jobs London

A premium web application developed for HireDev Ltd., a UK-based tech recruitment company. The design communicates authoritative professionalism, editorial warmth, and premium transformation using a crafted visual language (Tailwind CSS, Framer Motion) focused on high-conversion and prestige.

## Tech Stack
* **Framework:** Vite + React (v19)
* **Routing:** React Router v7
* **Styling:** Tailwind CSS v4 (Custom brand tokens via CSS variables)
* **Animations:** Framer Motion
* **Icons:** Lucide React

## Design Directives
* **Palette:** HireDev Teal (`#1C5C5C`), Warm Parchment (`#F5F0E8`), Gold (`#E8C547`), Deep Teal (`#0D1F1F`)
* **Typography:** Cormorant Garamond (Headlines, Editorial), DM Sans (UI, Clean readability), DM Mono (Stats, Tech context)
* **Aesthetics:** Glassmorphism accents, cinematic hero scroll reveals, soft subtle noise overlays, precision motion.

## Local Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Key Deliverables Implemented
* **Component-based structure:** Reusable layouts (Navbar, Footer, WhatsApp Button, etc.).
* **All 5 Core Pages Built:**
  - `Home`: Animated hero, trust bar, tiered steps, social proof carousels.
  - `About`: Company story, embedded impact stats, value pillars.
  - `Roles`: Dynamic tab switching for Technical vs. Business pathways.
  - `Placements`: Story grids, rich placement cards, video mockup embeds.
  - `Contact`: Multi-column contact form, direct paths, mapped integration.
* **SEO & Metadata:** Implemented `<title>` tags, descriptions within `index.html`.
